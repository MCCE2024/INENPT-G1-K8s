Die Customer Journey

  1. Code Push: Entwickler committen zu INENPT-G1-Code
  2. CI/CD: Automatischer Build & Test, Images landen im Registry
  3. Infrastruktur: INENPT-G1-K8s provisioniert und verwaltet die Cloud
  4. GitOps: INENPT-G1-Argo deployed und synchronisiert alles automatisch
Keine manuellen Schritte. Keine Downtime. 100% Nachvollziehbarkeit.
